MNISTClustering¶

Le clustering ou regroupement automatique permet de trouver des structures de groupes dans les donnes. Ce notebook prsente diffrents algorithmes en utilisant le jeu de donnes trs classique MNIST. Ce jeux de donnes contient des chiffre manuscrit numriss sous forme d'images de 8 pixels par 8 pixels. Nous allons mettre en oeuvre les trois algorithme de clustering vue en cours sur ce jeux de donnes grce la libraririe scikit-learn. Et nous visualiserons trs rapidement les rsultats obtenus. Nous allons commencer en chargeant les librairies dont nous avons besoin et en chargeant le jeu de donnes.

In [67]:
def plot_data(centers,vmin,vmax):
    f, axarr = plt.subplots(1,centers.shape[0], sharex=True,sharey=True)
    for i in range(centers.shape[0]):
        axarr[i].imshow(centers[i,].reshape([8,8]),cmap='Greys',clim=(vmin, vmax),interpolation='nearest')
        axarr[i].set( aspect='equal')
        axarr[i].get_xaxis().set_visible(False)
        axarr[i].get_yaxis().set_visible(False)

K-means¶

Nous avons charg la classe KMeans, nous pouvons donc maintenant dfinir une instance avec K=10 pour viter de tomber dans un minima local nous lancerons 10 fois l'algorithme (n_init=10). Pour xcuter l'lagorithme il suffit d'utiliser la mthode fit qui prend comme arguments les donnes clusteriser.

In [69]:
kmeans = KMeans(init='k-means++', n_clusters=n_digits, n_init=10)
kmeans.fit(digits.data)
Out[69]:
KMeans(algorithm='auto', copy_x=True, init='k-means++', max_iter=300,

Nous pouvons maintenant comparer les clusters obtenus aux "vrais" tiquettes du jeux de donnes pour nous faire une ide.
